# Hero Background Music Files

Place hero-specific background music files in this folder.

## File Naming Convention

Each hero should have a corresponding audio file:

- `flash.mp3` - Flash theme music
- `spider-man.mp3` - Spider-Man theme music
- `batman.mp3` - Batman theme music
- `aquaman.mp3` - Aquaman theme music
- `ant-man.mp3` - Ant-Man theme music
- `doctor-strange.mp3` - Doctor Strange theme music
- `wonder-woman.mp3` - Wonder Woman theme music
- `cid-moosa.mp3` - CID Moosa theme music
- `minnal-murali.mp3` - Minnal Murali theme music

## Audio Format

- Format: MP3 (preferred) or WAV
- Duration: 3-10 seconds recommended
- Quality: 128kbps or higher

## Adding New Heroes

When adding a new hero, add their theme music file following the naming pattern:
`{hero-name-lowercase-with-hyphens}.mp3`

For example: `super-man.mp3`
